동기코드:
모든 기기에서 같은 동기코드를 입력하면 데이터가 자동으로 동기화됩니다.
PC에서 입력한 내용이 모바일에 즉시 반영됩니다.
앱을 새로 열 때 자동으로 최신 데이터를 가져옵니다. 데이터가 반영 안 되면 위 버튼을 눌러주세요.
📋 준비 사항
1. Firebase 콘솔에서 프로젝트 선택 (기존 프로젝트 재사용 가능)
2. Firestore Database 활성화 확인
3. 프로젝트 설정 → 웹 앱 → apiKey / projectId 확인
아래 두 값만 입력하면 됩니다.
